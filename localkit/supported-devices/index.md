# Supported Devices

Currently, there are only a limited set of Petkit Devices supported. Every device needs to be seperate implemented.

::: info
To work properly, the devices needs a manipulated firmware, for the non-camera devices, this is possible via OTA.
:::

## [Petkit Pura Max](./pura-max)
![Petkit Pura Max](../public/pura-max.png)

Supported:
- K3 Spray
- Manual Control

## [Petkit Elements Solo](./elements-solo)
![Petkit Elements Solo](../public/petkit-elements-solo.png)

Supported:
- Schedules
- Manual Control


## [Petkit Yumshare Solo](./yumshare-solo)
![Petkit Yumshare Solo](../public/yumshare-solo.png)

Supported:
- Schedules
- Manual Control
- Camera Stream
